# Customer Support Ticket Dataset

This app is designed around the Kaggle Customer Support Ticket Dataset by `suraj520`:

https://www.kaggle.com/datasets/suraj520/customer-support-ticket-dataset

Before committing or deploying the full raw dataset, verify the Kaggle license and redistribution terms.

For local development, place the Kaggle CSV at:

```text
data/kaggle/customer_support_tickets.csv
```

If that file is not present, the seed script uses `customer_support_tickets.sample.csv`.
